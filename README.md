# ✦ ToDo App — Интерактивный список задач

**Современное CRUD-приложение с фильтрацией, сортировкой, Drag & Drop и тёмной темой**

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| ➕ **Добавление** | Создание задач с валидацией (пустые строки, лимит 200 символов) |
| ✅ **Статус** | Переключение выполнено/не выполнено с анимацией |
| 🗑 **Удаление** | Плавное удаление с анимацией slide-out |
| 🔍 **Фильтрация** | Все / Активные / Завершённые с счётчиками |
| 📊 **Сортировка** | По дате (новые/старые) + ручной порядок |
| ✋ **Drag & Drop** | Нативный HTML5 API — перетаскивание задач без библиотек |
| 🌙 **Тёмная тема** | Автоопределение + ручное переключение |
| 💾 **Persistence** | Данные сохраняются в localStorage |
| 📱 **Адаптивность** | Отлично выглядит на любом устройстве |

## 🛠 Технологии

- **React 19** + **TypeScript** — типобезопасный UI
- **Vite** — молниеносная сборка
- **Tailwind CSS 4** — утилитарные стили
- **HTML5 Drag & Drop API** — нативное перетаскивание без библиотек

## 🏗 Архитектура

```
src/
├── components/          # Презентационные компоненты
│   ├── AddTask.tsx      # Форма добавления с валидацией
│   ├── TaskItem.tsx     # Карточка задачи (memo-оптимизирована)
│   ├── TaskList.tsx     # Список + Drag & Drop логика
│   ├── FilterBar.tsx    # Фильтры + сортировка
│   ├── Header.tsx       # Заголовок + счётчик
│   └── ThemeToggle.tsx  # Переключатель темы
├── hooks/               # Бизнес-логика
│   ├── useTodos.ts      # Основной хук (CRUD + фильтрация)
│   └── useLocalStorage.ts # Персистентное хранение
├── types/               # TypeScript типы
│   └── todo.ts
├── utils/               # Утилиты
│   └── helpers.ts       # ID, валидация, форматирование

```

### Ключевые архитектурные решения:

- **Custom Hooks** — вся бизнес-логика вынесена в `useTodos`, компоненты остаются чистыми
- **Мемоизация** — `React.memo` для `TaskItem`, `useMemo` для фильтрации, `useCallback` для обработчиков
- **CSS Custom Properties** — темизация через CSS-переменные, мгновенное переключение
- **Optimistic UI** — анимация удаления до фактического удаления из state

## 🚀 Быстрый старт

```bash
# Клонировать
git clone https://github.com/your-username/todo-app.git
cd todo-app

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Запустить тесты
npm test

# Собрать для продакшена
npm run build
```
